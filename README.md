# Bootloader

This project builds a bootloader. I'm not sure why yet.

## Prerequisites

* nasm - The assembler that will be used to build the code
* qemu - The CPU emulator
* make - The build tool

## How to build it

```bash
make
```

## To run

```bash
make run
```
